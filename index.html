Visión General

Comprensión del Sistema

Este proyecto implementa un sistema de gestión para un avión de aerolínea con 50 sillas: 8 de clase ejecutiva y 42 de clase económica. El sistema permite asignar pasajeros a sillas según sus preferencias de ubicación, manteniendo un registro ordenado de las asignaciones.

Entidades Principales

  • Pasajero: Contiene información personal del viajero
  • Silla: Representa una silla en el avión con propiedades específicas
  • Avión: Gestiona el conjunto de sillas y operaciones de asignación

Clase

Pasajero

La clase Pasajero representa a una persona que viajará en el avión. Almacena información básica como la cédula y el nombre del pasajero.

Implementación

La implementación de esta clase es sencilla, con atributos privados y métodos para acceder a ellos:

Python pasajero.py 
class Pasajero:
    #---------------------------
    # Atributos
    #---------------------------
    def __init__(self, pCedula, pNombre):
        """
        Constructor de la clase Pasajero
        Parámetros:
        pCedula: Cédula del pasajero (tipo: long/int)
        pNombre: Nombre del pasajero (tipo: string)
        """
        self.__cedula = pCedula
        self.__nombre = pNombre
    
    #---------------------------
    # Métodos
    #---------------------------
    def darCedula(self):
        """
        Retorna la cédula del pasajero
        Returns: La cédula del pasajero
        """
        return self.__cedula
    
    def darNombre(self):
        """
        Retorna el nombre del pasajero
        Returns: El nombre del pasajero
        """
        return self.__nombre

Cosas a tener en cuenta

  • Utiliza el doble guión bajo (__) para hacer los atributos privados
  • Implementa los métodos métodos solicitados: darCedula() y darNombre()

Silla

La clase Silla representa las sillas disponibles en el avión. Cada silla tiene propiedades como su número, clase (ejecutiva o económica), ubicación (ventana, centro o pasillo) y una referencia al pasajero asignado, si existe.

Enumeraciones

Define tipos de clase y ubicación usando enumeraciones para mayor claridad y mantenibilidad

Asignación

Permite asignar y desasignar pasajeros manteniendo la integridad del sistema

Estado

Proporciona métodos para consultar el estado actual de la silla

Implementación

Esta clase es más compleja y utiliza enumeraciones para definir los tipos de clase y ubicaciones posibles:

Python silla.py 
from enum import Enum

class Silla:
    #------------------------------------------
    # Enumeraciones
    #------------------------------------------
    """
    * Enumeradores para las clases de silla.
    """
    class Clase(Enum):
        """
        * Representa la clase ejecutiva.
        """
        EJECUTIVA = 1
        
        """
        * Representa la clase económica.
        """
        ECONOMICA = 2
    
    """
    * Enumeradores para las ubicaciones de las sillas.
    """
    class Ubicacion(Enum):
        """
        * Representa la ubicación ventana.
        """
        VENTANA = 1
        
        """
        * Representa la ubicación centro.
        """
        CENTRO = 2
        
        """
        * Representa la ubicación pasillo.
        """
        PASILLO = 3
    
    #------------------------------------------
    # Atributos
    #------------------------------------------
    
    def __init__(self, pNumero, pClase, pUbicacion):
        """
        Constructor de la clase Silla
        Parámetros:
        pNumero: Número de la silla (tipo: int)
        pClase: Clase de la silla (tipo: Clase)
        pUbicacion: Ubicación de la silla (tipo: Ubicacion)
        """
        self.__numero = pNumero
        self.__clase = pClase
        self.__ubicacion = pUbicacion
        self.__pasajero = None
    
    def asignarPasajero(self, pPasajero):
        """
        Asigna la silla al pasajero "pPasajero".
        Parámetros:
        pPasajero: El pasajero a asignar (tipo: Pasajero)
        """
        self.__pasajero = pPasajero
    
    def desasignarSilla(self):
        """
        Quita al pasajero que se encuentra en la silla, dejándola desocupada.
        """
        self.__pasajero = None
    
    def sillaAsignada(self):
        """
        Informa si la silla está ocupada.
        Returns: True si la silla está ocupada, False en caso contrario
        """
        return self.__pasajero is not None
    
    def darNumero(self):
        """
        Retorna el número de la silla
        Returns: El número de la silla
        """
        return self.__numero

Aspectos destacados

  • Utiliza enum.Enum para crear enumeraciones tipo Java en Python
  • Mantiene atributos privados para encapsular el estado interno
  • Proporciona métodos específicos para la asignación y desasignación de pasajeros
  • El constructor inicializa __pasajero como None para representar una silla vacía

Avión

La clase Avión es el componente principal que gestiona las sillas y las operaciones de reserva en el sistema. Contiene una colección de sillas organizadas según su clase (ejecutiva o económica) y ubicación (ventana, centro, pasillo), y proporciona métodos para la gestión de reservas.

Configuración

Inicializa un avión con 50 sillas: 8 ejecutivas y 42 económicas, organizadas según ubicaciones específicas

Búsqueda

Proporciona métodos para buscar sillas disponibles según criterios específicos

Gestión

Implementa funcionalidades de asignación, anulación y consulta de reservas

Implementación

Esta clase coordina la gestión de todas las sillas y sus reservas, implementando métodos específicos para cada requerimiento:

Python avion.py 
from Silla import Silla
from Pasajero import Pasajero

class Avion:
    #------------------------------------------
    # Atributos
    #------------------------------------------
    def __init__(self):
        # Crear la matriz de sillas
        self.__sillas = []
        
        for i in range(1, 9):
            # Definir ubicación:
            # - Ventana: sillas 1, 4, 5, 8
            # - Pasillo: sillas 2, 3, 6, 7
            if i in [1, 4, 5, 8]:
                ubicacion = Silla.Ubicacion.VENTANA
            else:
                ubicacion = Silla.Ubicacion.PASILLO
            
            self.__sillas.append(Silla(i, Silla.Clase.EJECUTIVA, ubicacion))
        
        # Crear sillas económicas (9-50)
        # 7 filas con 6 sillas cada una (2 ventanas, 2 centros, 2 pasillos por fila)
        for i in range(9, 51):
            posicion = (i - 9) % 6 + 1
                        
            if posicion in [1, 6]:
                ubicacion = Silla.Ubicacion.VENTANA
            elif posicion in [2, 5]:
                ubicacion = Silla.Ubicacion.CENTRO
            else:  # posiciones 3 y 4
                ubicacion = Silla.Ubicacion.PASILLO
            
            self.__sillas.append(Silla(i, Silla.Clase.ECONOMICA, ubicacion))
    
    #------------------------------------------
    # Métodos del Taller clase 2
    #------------------------------------------
    
    # 1. Contar sillas ejecutivas ocupadas
    def contarSillasEjecutivasOcupadas(self):
        contador = 0
        for i in range(8):
            if self.__sillas[i].sillaAsignada():
                contador += 1
        return contador
    
    # 2. Buscar pasajero ejecutivo por cédula
    def buscarPasajeroEjecutivo(self, pCedula):
        for i in range(8):
            silla = self.__sillas[i]
            if silla.sillaAsignada() and silla.darPasajero().darCedula() == pCedula:
                return silla
        return None
    
    # 3. Buscar silla económica libre según ubicación
    def buscarSillaEconomicaLibre(self, pUbicacion):
        for i in range(8, len(self.__sillas)):
            silla = self.__sillas[i]
            if not silla.sillaAsignada() and silla.darUbicacion() == pUbicacion:
                return silla
        return None
    
    # 4. Asignar silla económica
    def asignarSillaEconomica(self, pUbicacion, pPasajero):
        silla = self.buscarSillaEconomicaLibre(pUbicacion)
        if silla is not None:
            silla.asignarPasajero(pPasajero)
            return True
        return False
    
    # 5. Anular reserva ejecutivo
    def anularReservaEjecutivo(self, pCedula):
        silla = self.buscarPasajeroEjecutivo(pCedula)
        if silla is not None:
            silla.desasignarSilla()
            return True
        return False
    
    # 6. Contar ventanas económicas disponibles
    def contarVentanasEconomica(self):
        contador = 0
        for i in range(8, len(self.__sillas)):
            silla = self.__sillas[i]
            if not silla.sillaAsignada() and silla.darUbicacion() == Silla.Ubicacion.VENTANA:
                contador += 1
        return contador
    
    # 7. Buscar homónimos en clase económica
    def hayDosHomonimosEconomica(self):
        nombres = []
        for i in range(8, len(self.__sillas)):
            silla = self.__sillas[i]
            if silla.sillaAsignada():
                nombre = silla.darPasajero().darNombre()
                # Si el nombre ya está en la lista, hay homónimos
                if nombre in nombres:
                    return True
                nombres.append(nombre)
        return False
    
    #------------------------------------------
    # Métodos de la Tarea casa 3
    #------------------------------------------
    
    # 1. Contar sillas económicas libres
    def contarSillasEconomicasLibres(self):
        contador = 0
        for i in range(8, len(self.__sillas)):
            if not self.__sillas[i].sillaAsignada():
                contador += 1
        return contador
    
    # 2. Contar pasillos ejecutivos disponibles
    def contarPasilloEjecutivas(self):
        contador = 0
        for i in range(8):
            silla = self.__sillas[i]
            if not silla.sillaAsignada() and silla.darUbicacion() == Silla.Ubicacion.PASILLO:
                contador += 1
        return contador
    
    # 3. Desocupar avión
    def desocuparAvion(self):
        for i in range(len(self.__sillas)):
            self.__sillas[i].desasignarSilla()

Aspectos destacados

  • La clase Avión gestiona la configuración física de las sillas del avión, definiendo sus posiciones y características
  • El constructor inicializa la distribución del avión con 8 sillas ejecutivas y 42 económicas
  • Implementa métodos específicos para contar, buscar y gestionar sillas según diferentes criterios
  • Utiliza encapsulamiento para proteger el atributo __sillas y proporciona métodos para la interacción con ellas
  • Los métodos se dividen en dos grupos: los del "Taller clase 2" y los de la "Tarea casa 3", mostrando una organización clara del código
